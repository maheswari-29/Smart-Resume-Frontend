This folder contains the frontend part of the Smart Resume Analyzer application.

Features

Resume upload with drag and drop

Job description input

Match score display

Matched and missing skills display

Dark mode support

Responsive layout

Technologies Used

HTML5

CSS3

JavaScript

How to Run the Frontend
Option 1: Open Directly

Open index.html in a web browser.

Option 2: Live Server (Recommended)

Install Live Server extension in VS Code

Right-click index.html

Click Open with Live Server

Backend Configuration

Update the backend URL in index.js if needed:

const BACKEND_URL = "https://your-backend-url.onrender.com";

Notes

Frontend works as a static site

Backend must be running for analysis to work
